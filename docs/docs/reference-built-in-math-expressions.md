Adama lets you do math, and that's awesome! It has the typical operations that you would expect, and then some fun twists. So, let's get into it.

## Parenthesis: ()
## Unary Negation: -, !
## Addition: +

## Subtraction: -

## Multiplication: *
## Division: /

## Modulus: %

## Comparison: <, <=, >, >=

## Equality: ==, !=

## Logic: &&, ||
## Conditional: ?
